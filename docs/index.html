Background colors
======================
Use .bg-COLOR-N to set background color.
COLOR = gray, red, green, blue, orange, teal, purple, yellow, pink, coral
N = 0 to 9
for example-
.bg-red-9
.bg-0 = transparent

Foreground colors
======================
Use .c-COLOR-N to set foreground color.
COLOR = gray, red, green, blue, orange, teal, purple, yellow, pink, coral
N = 0 to 9
for example-
.c-red-9
.c-0 = transparent

Set Border colors
======================
Use .bc-COLOR-N to set border color.
COLOR = gray, red, green, blue, orange, teal, purple, yellow, pink, coral
N = 0 to 9
for example-
.bc-red-9
.bc-0 = transparent

Margins
==================================
Syntax: .xx-yyy[-zz] xx = ma/mt/mb/ml/mr/mv/mh, yyy = auto/0/025/050/075/100/125/150/175/200/225/250/275/300/400, zz = sm/lg
ma: margin all
mt: margin-top
mb: margin-bottom
ml: margin-left
mr: margin-right
mv: vertical margin (top+bottom)
mh: horizontal margin (left+right)
Example
-------
.ma-auto : creates 'margin: auto' on all devices.
.ma-auto-sm : creates 'margin: auto' on small devices.
.ma-auto-lg : creates 'margin: auto' on large devices.
.mb-025 : creates 'margin-bottom: 0.25rem' on all devices.
.mb-025-sm : creates 'margin-bottom: 0.25rem' on small devices.
.mb-025-lg : creates 'margin-bottom: 0.25rem' on large devices.

Padding

pa: padding all
pt: padding-top
pb: padding-bottom
pl: padding-left
pr: padding-right

ph: horizontal padding (left+right)

Example

Hover Effect
============
.hover:bright-120% : Increase 120% brightness on hover. Other values are- 100,120,140,160,180,200,300,400
.hover:dark-80% : Decrease 80% brightness on hover. Other values are- 80,60,40,20,0
